#使用说明

第一步编译：
修改Makefile ，根据内核编译
$make
第二步编译应用程序：
$arm-xilinx-linux-gnueabi-gcc -o registor registor.c

第三步 使用：
#ls
ker_rw.ko  regeditor
# chmod +x regeditor 
# insmod ker_rw.ko
